House Price Prediction Model

Linear Regression is often the gateway into the predictive capability of artificial intelligence. In this project, I used real-world real estate datasets to build a machine learning model that forecasts housing valuations.

First, I used Pandas to clean columns and handle missing data points, and NumPy to perform fast vector operations. Using Scikit-Learn, I fit a multi-variable Linear Regression model, calculating optimal weights for space square footage, bedroom density, and location tier indices.
